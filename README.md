# BioPrint AI

## Fingerprint-Based Blood Group Detection through Artificial Intelligence, Image Processing, and Machine Learning

BioPrint AI is an innovative healthcare-focused artificial intelligence project that explores the possibility of predicting probable blood groups through fingerprint image analysis. By combining biometric science, computer vision, and machine learning, this system presents a research-oriented, non-invasive prototype designed to demonstrate how modern AI technologies can contribute to future healthcare accessibility.

Built using OpenCV, Streamlit, and Random Forest classification, BioPrint AI transforms fingerprint images into meaningful biometric insights through ridge density analysis, minutiae approximation, and predictive modeling.

## Project Vision

Traditional blood group identification methods depend on blood samples, laboratory infrastructure, and trained medical professionals. While highly accurate, these methods can be time-consuming and invasive.

BioPrint AI explores an alternative academic and research-based concept by analyzing fingerprint characteristics to build a machine learning model capable of predicting blood groups probabilistically. This project represents the intersection of healthcare innovation and artificial intelligence, showcasing how biometric systems may contribute to future smart diagnostic support tools.

## Key Features

## Artificial Intelligence Workflow

### Image Processing Layer
The uploaded fingerprint image undergoes:
- Grayscale conversion
- Noise reduction
- Thresholding
- Ridge enhancement

### Feature Extraction Layer
The system extracts:
- Ridge Density
- Minutiae Count
- Fingerprint Pattern Features

### Machine Learning Layer
- Random Forest Classifier for blood group prediction
