# BioPrint AI

## Fingerprint-Based Blood Group Detection using Image Processing and Machine Learning

BioPrint AI is a research-oriented biometric healthcare project designed to predict probable blood groups through fingerprint image analysis using OpenCV and Random Forest classification.

## Features
- Fingerprint image upload
- Ridge density analysis
- Minutiae count approximation
- Blood group prediction
- Confidence score
- Probability distribution dashboard

## Technology Stack
- Python
- Streamlit
- OpenCV
- NumPy
- Pandas
- Scikit-learn
- Pillow
